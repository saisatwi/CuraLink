# 🧬 CuraLink - AI Medical Research Assistant

**Hackathon Submission** for Curalink AI Medical Research Assistant.

[![Live Demo](https://img.shields.io/badge/Demo-Vercel-blue?style=for-the-badge&logo=vercel)](https://curalink-rouge.vercel.app/)

---

## 🛠️ Tech Stack

| Layer         | Technology                    | Role |
|---------------|-------------------------------|------|
| Frontend      | React.js + Tailwind CSS       | Modern Chat Interface |
| **AI Engine** | **Ollama (llama3.2)**         | Local Open-Source LLM (Hackathon Compliant) |
| Backend       | Node.js + Express             | Research Orchestrator |
| Database      | MongoDB Atlas                 | Conversation History |
| Data Sources  | PubMed + OpenAlex + ClinicalTrials.gov | Real Research Retrieval |
| Hosting       | Vercel (Frontend) + Render (Backend) | Live Deployment |

---

## ✨ Key Features
- Structured Input (Disease + Query)
- Deep retrieval from PubMed, OpenAlex & ClinicalTrials.gov
- Source-backed responses with DOI, PMID, Authors, Year
- Context-aware multi-turn conversation
- Fully local LLM (Ollama)
